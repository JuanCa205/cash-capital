# 🎯 Cash Capital - App de Gamificación Económica

Una aplicación educativa que combina aprendizaje financiero con gamificación.

## 🚀 Características Principales

- 📚 Módulos de aprendizaje interactivos
- 💰 Planificador de finanzas personales  
- 🎮 Sistema de gamificación con puntos y niveles
- 👥 Funcionalidades sociales y competencias

## 🛠️ Stack Tecnológico

- **Frontend:** React + TypeScript + Vite + TailwindCSS
- **Backend:** Node.js + Express + PostgreSQL
- **DevOps:** Docker + Docker Compose
